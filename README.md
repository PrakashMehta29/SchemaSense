# 📊 SchemaSense

An AI-powered intelligence platform that breaks down technical and language barriers in data analysis by translating complex database schemas into clear, human-readable insights.

---

## 🌟 Core Features

- **Dynamic Schema Analysis:** Instantly processes technical database structures into high-level visual summaries.
- **Multilingual Support:** Breaks global communication barriers by allowing users to explore and interact with data configurations in multiple regional languages.
- **Voice-Enabled Accessibility:** Integrated **Talk-to-Speech** features for intuitive, hands-free data exploration.
- **User-Centric UI:** Designed with a clean dashboard layout transforming complex system setups into interactive, readable structures.

---

## 🛠️ Built With

- **Frontend:** React, TypeScript, Tailwind CSS
- **Development Tools:** Bun, Vite

---

## 🚀 Getting Started

### Prerequisites

Make sure you have [Bun](https://bun.sh/) installed on your local machine.

### Installation & Local Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/PrakashMehta29/SchemaSense.git](https://github.com/PrakashMehta29/SchemaSense.git)
   ```
2. **Navigate into the project directory:**
   ```bash
   cd SchemaSense
   ```
3. **Install dependencies:**
   ```bash
   bun install
   ```
4. **Start the local development server:**

   ```bash
   bun run dev

   ```

5. **Clone the repository:**
